The code is written by Yang Pu at the Northwestern Polytechnical Univeristy.

MTBO is a constraint-aware Bayesian optimization framework that enables hierarchical, multi-objective process optimization for perovskite solar cell fabrication.

Some key packages (and their versions) required to run this code are as follows:

numpy 1.22.0

pandas 1.5.3

Chimera 1.0

emukit 0.4.9

matplotlib 3.4.1
